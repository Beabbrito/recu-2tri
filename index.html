Os principais comandos para se criar um site

Venha descobrir os principais comandos que são usados para criar em sites.

O TITLE define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

O HEAD, é a cabeça de todo site, é a parte que não é exibida no navegador quando a página é carregada.

Os "h" são os títulos, representam seis níveis de título de seção. O h1 é o nível de seção mais alto e h6 é o mais baixo.

o BODY é o corpo do site, representa o conteúdo de um documento HTML.

O HTML é o que faz o site funcionar, tudo fica dentro dele
